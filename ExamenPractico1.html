Veterinaria

Descripición del problema

  • Un conocido hospital veterinario, está solicitando el desarrollo de una aplicación web para la utilización de su personal y de sus clientes. Desarrolla el diagrama E-R, el diagrama relacional, y el modelo físico, según las siguientes consideraciones:
  • El hospital atiende diferentes tipos de mascotas, perros, gatos, aves, entre otras especies. Cada cliente puede acudir con diferentes especies de mascotas. Los servicios del hospital veterinario pueden ser consulta, urgencia, estética, venta de alimento, venta de accesorios
  • En el hospital atienden 3 médicos veterinarios en los diferentes turnos.
  • Como evidencia, enviar: Diagrama E-R. Diagrama Relacional (normalizado) Base de datos (incluyendo las consultas MySQL y la estructura de las tablas). Dirección URL del examen publicado en su hosting web.
